# Landing Page - Dra. Izandra

Landing page profissional para a Dra. Izandra, advogada especializada em Direito de Família e Sucessões.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Font Awesome
- Google Fonts

## 📋 Características

- Design moderno e profissional
- Totalmente responsivo
- Animações suaves
- Formulário de contato
- Botão de WhatsApp flutuante
- Seções bem organizadas
- Otimizado para SEO

## 🎨 Paleta de Cores

- Azul Marinho (Primary): #1a237e
- Vinho (Secondary): #8b0000
- Dourado (Accent): #c5a47e
- Texto: #333
- Fundo Claro: #f8f9fa

## 📱 Responsividade

A landing page é totalmente responsiva e se adapta aos seguintes dispositivos:
- Desktop
- Tablet
- Mobile

## 🔧 Instalação

1. Clone este repositório
2. Abra o arquivo `index.html` em seu navegador

## 📦 Estrutura de Arquivos

```
├── index.html
├── css/
│   └── style.css
├── js/
│   └── main.js
└── img/
    ├── advogada.jpg
    └── sobre.jpg
```

## 🛠️ Personalização

Para personalizar a landing page:

1. Substitua as imagens em `img/` com suas próprias fotos
2. Atualize as informações de contato no arquivo `index.html`
3. Modifique as cores no arquivo `css/style.css` (variáveis CSS)
4. Atualize os textos e conteúdos conforme necessário
